# KinopoiskApi
## How can you run it?
### 1. Clonning repo
   ```
   git clone https://github.com/nastyazz/KinopoiskApi
   ```

### 2. Create Free API token in Telegram: @kinopoiskdev_bot

### 3. In the file config.py, change the token to yours
   ```
   KINOTOKEN = os.getenv('KINOTOKEN', 'YOUR_TOKEN')
   ```
   
### 4. Run app
   ```
   cd myproject
   python3.11 app.py
   ```

### 5. Path to the site
   ```
     http://127.0.0.1:8080
   ```
